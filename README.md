# La Cigale — Brasserie Mythique à Nantes

Bienvenue dans le projet de refonte frontend de **La Cigale**. Ce projet est une implémentation React haute-fidélité basée sur les maquettes Stitch et les spécifications Art Nouveau du monument nantais.

## 🚀 Lancement Rapide

Pour lancer l'application en mode développement :

1.  **Installation des dépendances** (si ce n'est pas déjà fait) :
    ```bash
    npm install
    ```

2.  **Démarrage du serveur de développement** :
    ```bash
    npm run dev
    ```

3.  **Accès direct** :
    L'application sera disponible sur `http://localhost:5173`.

## 🛠 Stack Technique

- **Framework** : React + TypeScript
- **Build Tool** : Vite
- **Styling** : Tailwind CSS v4 (Alpha/Experimental integration)
- **Typography** : Cormorant Garamond (Display) & Montserrat (Body)
- **Icons** : Material Icons

## 📂 Structure du Projet

- `/src/components/common` : Éléments d'interface réutilisables (Header, Footer, Layout).
- `/src/components/home` : Sections spécifiques de la page d'accueil (Hero, Experience, Table, etc.).
- `/src/pages` : Pages principales de l'application.
- `/src/styles` : Configuration globale CSS et thèmes.
- `/docs` : Spécifications UI et plans d'architecture.

## 🛡 Qualité & QA

Un fichier `agents_correcteur.md` est présent à la racine pour guider les audits QA et assurer un alignement **Pixel-Perfect** avec les maquettes de référence.
